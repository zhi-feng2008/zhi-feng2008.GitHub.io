Fenix A320 导航数据安装教程

当前最新版本为 : 2308

安装步骤

  1. 打开下载文件夹并解压
  2. 将以下导航数据文件复制到Fenix导航目录，并覆盖现有的文件：（以下均为默认路径）
    • 文件目录\Navdata\cycle_info.txt -> C:\ProgramData\Fenix\Navdata
    • 文件目录\Navdata\cycle.json -> C:\ProgramData\Fenix\Navdata
    • 文件目录\Navdata\nd.db3 -> C:\ProgramData\Fenix\Navdata
  3. 开始享受的飞行吧

注意：导航目录需要开启显示隐藏文件才能显示，安装完成后可通过MCDU查看是否生效
