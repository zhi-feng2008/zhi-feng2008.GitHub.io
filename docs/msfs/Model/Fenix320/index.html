Fenix A320 安装教程

  1. 运行并安装 FenixSimA320-1.0.6.146.exe
  2. 安装完成后，Fenix 应用程序将自动启动，请先退出
  3. 将以下和谐的可执行文件复制到你的的 Fenix A320 安装目录，并覆盖现有的文件：（以下均为默认路径）
    • 文件目录\Crack\Fenix.exe -> C:\Program Files\FenixSim A320
    • 文件目录\Crack\deps\FenixCDU.exe -> C:\Program Files\FenixSim A320\deps
    • 文件目录\Crack\deps\FenixDisplay.exe -> C:\Program Files\FenixSim A320\deps
    • 文件目录\Crack\deps\FenixSystem.exe -> C:\Program Files\FenixSim A320\deps
  4. 开始享受的飞行吧

注意：

在使用机模型时可能会出现一些奇怪的情况，有时重新启动飞行后仪表可能不再工作，或出现其他奇怪的行为。如果出现这种情况，请尝试重新启动 MSFS、Fenix 应用程序甚至是电脑。这主要是因为第三方程序在MSFS内的飞机，而不是在自己独立运行。这与和谐版本无关，正常版本也会出现类似问题。

有时候关闭MSFS后，“deps”文件夹中的可执行文件可能会卡住，如果出现这种情况，您可以在 Windows 任务管理器中结束它们的进程。
